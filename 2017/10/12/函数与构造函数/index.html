学习日记1--函数与构造函数的区别

函数与构造函数的区别
这个问题可以从普通函数调用与构造器调用时的区别来入手。而最大的区别就是在普通函数调用时其内部this指向了全局对象。
如果是浏览器中运行那么这个对象就是window。
如果用作构造器调用，如var a = new A(),此时A中的this会指向通过new调用出来的实例化对象。
例：
function A(name, id) {
this.name = name;
this.id = id;
}
function B(name, id) {
this.name = name;
this.id = id;
}
var a = new A(1,2);
var b = B(3,4);
console.log(a);// {id: 2, name: 1}, this === a
console.log(b);// undefined
console.log(window.name);// ‘3’

console.log(window.id);// 4

call&&apply&&bind调用
一定要记住！这三个方法的第一个传入的参数，将会是调用这个方法中的this的新指向
var obj = {
a: 1,
getA: function(){
console.log(this);
console.log(this.a);
}
}
var obj2 = {
a: 2;
}
obj.getA.call(obj2);// this === obj2, a === 2
看到了吗，调用call之后，与其第一个参数也就是obj2成为了调用者obj中this的新指向。
